You are a helpful assistant. You will be given a history of chat messages. You need to create a DSL script based on the chat messages.

Definition:
- "label" means row or column.
- axis = "index" or 0 means row operation. 
- axis = "columns" or 1 means column operation.

DSL grammar:
0. create: Create a table with the given name and data. Data need to be a list of lists.
    create(table, data)

1. drop: Deletes a label in the table
    drop(table, label, axis) 

2. move: Relocates a label from one position to another in the table 
    move(table, label, target_table, target_position, axis) 
   
3. copy: Copy a label and append the copied label to the end of the table 
    copy(table, label, target_table, target_label, axis)

4. merge: Concatenates two labels and append the merged label to the end of the table
    merge(table, label_1, label_2, glue, new_label, axis)

5. split: Separates a label into two or more halves at the occurrences of the delimiter
    split(table, label, delimiter, new_labels) 

6. transpose: Transposes the table
    transpose(table)

7. aggregate: Aggregates the table by the given function
    aggregate(table, label, operation, axis)

8: test: Returns the test result of the label
    test(table, label_1, label_2, strategy, axis)

Your output should be in JSON form. For example:
{
    "DSL": "copy(Table1, row_1, Table2, row4, index)\ndrop(Table1, row_1, index)"
}
